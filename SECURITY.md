# Security Policy

If you believe you have found a security vulnerability in this repository or in any deployed service,
please report it responsibly.

## How to report

Email: `security@example.com`

Please include:
- A clear description of the issue
- Steps to reproduce (if applicable)
- Impact and severity assessment
- Expected vs actual behavior
- Any relevant logs or screenshots (redact secrets)

## Response times

We will:
- Acknowledge your report within 2 business days
- Provide an initial triage within 5 business days

## Safe handling of reports

Please do not create public issues or PRs containing exploit details until we have confirmed
the scope and guidance for a fix.
